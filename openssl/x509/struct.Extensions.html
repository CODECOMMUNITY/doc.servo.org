struct Extensions {
    extensions: Vec<Extension>,
    indexes: HashMap<ExtensionType, usize>,
}

A collection of X.509 extensions.

Upholds the invariant that a certificate MUST NOT include more than one instance of a particular extension, according to RFC 3280 §4.2. Also ensures that extensions are added to the certificate during signing in the order they were inserted, which is required for certain extensions like SubjectKeyIdentifier and AuthorityKeyIdentifier.
