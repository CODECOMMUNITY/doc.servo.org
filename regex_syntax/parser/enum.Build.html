enum Build {
    Expr(Expr),
    LeftParen {
        i: CaptureIndex,
        name: CaptureName,
        chari: usize,
        old_flags: Flags,
    },
}

An ephemeral type for representing the expression stack.

Everything on the stack is either a regular expression or a marker indicating the opening of a group (possibly non-capturing). The opening of a group copies the current flag state, which is reset on the parser state once the group closes.
