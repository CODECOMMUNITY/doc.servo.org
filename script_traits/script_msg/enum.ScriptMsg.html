Messages from the script to the constellation.

Variants

ChangeRunningAnimationsState

Indicates whether this pipeline is currently running animations.

CreateCanvasPaintThread

Requests that a new 2D canvas thread be created. (This is done in the constellation because 2D canvases may use the GPU and we don't want to give untrusted content access to the GPU.)

CreateWebGLPaintThread

Requests that a new WebGL thread be created. (This is done in the constellation because WebGL uses the GPU and we don't want to give untrusted content access to the GPU.)

DOMLoad

Dispatched after the DOM load event has fired on a document Causes a load event to be dispatched to any enclosing frame context element for the given pipeline.

Focus

Notifies the constellation that this frame has received focus.

ForwardMouseButtonEvent

Re-send a mouse button event that was sent to the parent window.

ForwardMouseMoveEvent

Re-send a mouse move event that was sent to the parent window.

GetClipboardContents

Requests that the constellation retrieve the current contents of the clipboard

HeadParsed

tag finished parsing

LoadComplete

All pending loads are complete.

LoadUrl

A new load has been requested.

MozBrowserEvent

Dispatch a mozbrowser event to a given iframe. Only available in experimental mode.

Navigate

HTMLIFrameElement Forward or Back navigation.

NewFavicon

Favicon detected

NodeStatus

Status message to be displayed in the chrome, eg. a link URL on mouseover.

RemoveIFrame

Notification that this iframe should be removed.

ScriptLoadedURLInIFrame

A load has been requested in an IFrame.

SetClipboardContents

Requests that the constellation set the contents of the clipboard

ActivateDocument

Mark a new document as active

SetDocumentState

Set the document state for a pipeline (used by screenshot / reftests)

SetFinalUrl

Update the pipeline Url, which can change after redirections.

Alert

Check if an alert dialog box should be presented

ScrollFragmentPoint

Scroll a page in a window

SetTitle

Set title of current page https://html.spec.whatwg.org/multipage/#document.title

SendKeyEvent

Send a key event

GetClientWindow

MoveTo

Move the window to a point

ResizeTo

Resize the window to size

TouchEventProcessed

Script has handled a touch event, and either prevented or allowed default actions.

GetScrollOffset

Get Scroll Offset

PipelineExited

Notifies the constellation that this pipeline has exited.

Exit

Requests that the compositor shut down.
