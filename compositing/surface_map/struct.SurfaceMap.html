This is a struct used to store surfaces when they are not in use. The paint thread can quickly query for a particular size of surface when it needs it.

Fields

map

A HashMap that stores the Buffers.

mem

The current amount of memory stored by the SurfaceMap's surfaces.

max_mem

The maximum allowed memory. Unused surfaces will be deleted when this threshold is exceeded.

counter

A monotonically increasing counter to track how recently tile sizes were used.
