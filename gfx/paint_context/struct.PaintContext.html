screen_rect

The rectangle that this context encompasses in screen coordinates (pixels).

clip_rect

The clipping rect for the stacking context as a whole.

transient_clip

The current transient clipping region, if any. A "transient clipping region" is the clipping region used by the last display item. We cache the last value so that we avoid pushing and popping clipping regions unnecessarily.

layer_kind

A temporary hack to disable clipping optimizations on 3d layers.

radius is the border-radius width and height. If radius.width == radius.height then an arc from a circle is drawn instead of an arc from an ellipse.

dist_elbow is the distance from origin to the inner part of the border corner. clockwise indicates direction to draw the border curve.
