Servo, the mighty web browser engine from the future.

This is a very simple library that wires all of Servo's components together as type Browser, along with a generic client implementing the WindowMethods trait, to create a working web browser.

The Browser type is responsible for configuring a Constellation, which does the heavy lifting of coordinating all of Servo's internal subsystems, including the ScriptThread and the LayoutThread, as well maintains the navigation context.

The Browser is fed events from a generic type that implements the WindowMethods trait.
