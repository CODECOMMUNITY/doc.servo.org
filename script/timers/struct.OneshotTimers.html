suspension_offset

Initially 0, increased whenever the associated document is reactivated by the amount of ms the document was inactive. The current time can be offset back by this amount for a coherent time across document activations.

expected_event_id

Calls to fire_timer with a different argument than this get ignored. They were previously scheduled and got invalidated when - timers were suspended, - the timer it was scheduled for got canceled or - a timer was added with an earlier callback time. In this case the original timer is rescheduled when it is the next one to get called.
