pub struct Root<T: Reflectable> {
    ptr: NonZero<*const T>,
    root_list: *const RootCollection,
}

A rooted reference to a DOM object.

The JS value is pinned for the duration of this object's lifetime; roots are additive, so this object's destruction will not invalidate other roots for the same JS value. Roots cannot outlive the associated RootCollection object.
