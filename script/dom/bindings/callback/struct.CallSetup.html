    old_compartment: *mut JSCompartment,

}

A class that performs whatever setup we need to safely make a call while this class is on the stack. After new returns, the call is safe to make.

Fields

exception_compartment

The compartment for reporting exceptions. As a RootedObject, this must be the first field in order to determine the final address on the stack correctly.

cx

The JSContext used for the call.

old_compartment

The compartment we were in before the call.

handling

The exception handling used for the call.
