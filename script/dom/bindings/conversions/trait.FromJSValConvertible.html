Associated Types

Optional configurable behaviour switch; use () for no configuration.

Required Methods

unsafe fn from_jsval(cx: *mut JSContext, val: Handle<Value>, option: Self::Config) -> Result<Self, ()>

Convert val to type Self. Optional configuration of type T can be passed as the option argument. If it returns Err(()), a JSAPI exception is pending.
